# txtTimer

Are you tired of the dozens of .txt files you’ve created on your desktop? Then this little system might come in handy.

---

### `del(“time in minutes”)`
This automatically deletes the text files you name once the specified number of minutes has elapsed.

**Example:**
```text
del(1).txt = delete this text file in 1 minute
or
del(30).txt = delete this text file in 30 minutes
